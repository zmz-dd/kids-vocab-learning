
# KET 一起学 (Vocab Kids) v4.0 🚀

一个为小朋友设计的、基于科学记忆法的英语背单词网页工具。

## ✨ 核心功能

- **艾宾浩斯记忆曲线**：智能算法安排复习计划，确保高效记忆。
- **自定义学习计划**：支持设定每日词数或目标完成天数，自动分配任务。
- **内置 KET 词库**：预设剑桥少儿英语一级、二级、三级共 1300+ 核心词汇，配有音标、例句和发音。
- **管理员工具**：专为家长/教师设计，支持一键上传单词列表并自动通过 API 补全详细释义。
- **趣味 PK 与头像**：内置多用户系统，通过“掌握量”排行榜激发学习热情，支持选择卡通头像。
- **详细统计**：可视化展示每个单词本的学习与复习进度。

## 🛠️ 技术栈

- **前端**：React 19 + TypeScript + Vite
- **UI 组件库**：Tailwind CSS v4 + shadcn/ui + Lucide Icons
- **动画**：Framer Motion
- **路由**：Wouter
- **数据持久化**：LocalStorage

## 🚀 快速开始

1. **克隆项目**: `git clone ...`
2. **安装依赖**: `pnpm install`
3. **启动开发服务器**: `pnpm dev`
4. **构建项目**: `pnpm build`
